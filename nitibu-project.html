Nitibu – Healthcare Appointment Platform

Nitibu is an innovative online healthcare appointment platform designed to simplify access to healthcare services. By leveraging advanced search functionalities and real-time location data, Nitibu connects patients with healthcare providers efficiently and reliably.

Core Features:

  • Authentication and Authorization: Secure user registration, login, and email verification.
  • Location-Based Search: Find doctors based on proximity to the user's current location.
  • Advanced Search Filters: Search by doctor name, symptoms, hospital/clinic name, or specialty.
  • Appointment Booking System: Real-time appointment scheduling and availability updates.
  • User Dashboard: Manage appointments, view booking history, and access notifications.
  • Doctor Profiles: Detailed doctor profiles with specialization, availability, and reviews.
  • Email Notifications: Automated email alerts for appointment confirmations and reminders.

Technology Stack:

Backend: Django, Django REST Framework, PostgreSQL.
Frontend: HTML5, CSS3, JavaScript.
APIs: JWT Authentication, Google Maps API for location services.
Infrastructure: Docker, AWS.

Architecture Design:

  • Modular Architecture: Authentication, Search, Booking, and User Services.
  • Scalability: Designed to handle increasing user loads efficiently.
  • Security: End-to-end encryption and secure authentication protocols.
